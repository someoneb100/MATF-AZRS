Dat je program koji je razvijao tim C++ početnika. Program učitava podatke iz input.txt, obrađuje ih i ispisuje 
rezultat na standardni izlaz. Program ne daje očekivani rezultat. Potrebno je uradite sledeće:
- Popravite program tako da daje očekivani rezultat (debagovanje);
- Koristite sistem za verzionisanje pri izmeni programa.
- Optimizujte kod koliko je to moguće (bonus deo).

Napomena: Svaki komit treba da sadrži kratko objašnjenje za modifikaciju programa.

-------------------------------------------------------------------------------------

'input.txt': Primer ulaza
'expected_result.txt': Primer ocekivanog izlaza za ulaz 'input.txt'
